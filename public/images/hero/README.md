Drop hero background media here — the homepage picks it up automatically, no code
changes needed.

- `hero-video.mp4` (or `.webm`) — a looping background video behind the hero text and
  animated flag. Takes priority over the background photo if both are present. Keep it
  short, muted-friendly, and not too large (a few MB, ideally under 10s).
- `hero-bg.jpg` (or `.jpeg` / `.png` / `.webp`) — a wide background photo (e.g. the
  Himalayas) used if no video is present, or as the video's poster frame.

This README is just documentation — delete it once you've added your files if you
like, it isn't referenced by the app.
