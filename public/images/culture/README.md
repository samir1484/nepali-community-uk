Drop culture/heritage photos here — the homepage's "Nepali Heritage & Culture" section
picks them up automatically by filename, no code changes needed. Until a file is
added, that slot shows a "Photo coming soon" placeholder.

Use any of `.jpg`, `.jpeg`, `.png`, or `.webp` with these exact names:

- `mountains.jpg` — the Himalayas
- `khukuri.jpg` — the Gurkha khukuri
- `stupa.jpg` — a Buddhist stupa (e.g. Boudhanath or Swayambhunath)
- `festival.jpg` — festivals and tradition (e.g. Dashain, Tihar)

Portrait-ish photos (roughly 4:5, taller than wide) will look best — that's the crop
shape used in the gallery cards.

This README is just documentation — delete it once you've added your files if you
like, it isn't referenced by the app.
